zensols.dbpg.postgres module¶

Postgres implementation of the ConnectionManager.

class zensols.dbpg.postgres.PostgresConnectionManager(db_name, host, port, user, password, create_db=True, capture_lastrowid=False, fast_insert=False)[source]¶

Bases: zensols.db.bean.ConnectionManager

An Postgres connection factory.

Parameters

  • persister – the persister that will use this connection factory (needed to get the initialization DDL SQL)

  • db_name (str) – database name on the server

  • host (str) – the host name of the database

  • port (str) – the host port of the database

  • user (str) – the user (if any) to log in with

  • password (str) – the login password

  • create_db (bool) – if True create the database if it does not already exist

  • capture_lastrowid (bool) – if True, select the last row for each query

  • fast_insert (bool) – if True use insertmany on the cursor for fast insert in to the database
